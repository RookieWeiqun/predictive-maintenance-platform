发现端点Discovery Endpoint
https://api.qa.oneid.siemens.com.cn/api/bff/v1.2/developer/ciam/oidc/6d641a7f3713a34d6c282baff235aefa6XFrDLLlFfN/.well-known/openid-configuration
用于获取当前 IDaaS 支持的各端点信息和支持的模式、参数信息，可公开访问。

授权端点Authorization Endpoint
https://api.qa.oneid.siemens.com.cn/api/bff/v1.2/developer/ciam/oauth/authorize
应用发起单点登录的地址。

令牌端点
https://api.qa.oneid.siemens.com.cn/api/bff/v1.2/developer/ciam/oidc/token
应用在单点登录过程中，拿到 code 后，从后端发去换取 token 的接口地址。

令牌吊销端点
https://api.qa.oneid.siemens.com.cn/api/bff/v1.2/developer/ciam/oidc/token/revoke

验签公钥端点
https://api.qa.oneid.siemens.com.cn/api/bff/v1.2/developer/ciam/oidc/6d641a7f3713a34d6c282baff235aefa6XFrDLLlFfN/.well-known/jwks.json
用于验证 id_token、完成 SSO 流程的公钥端点。公钥可能会轮转。

用户信息端点
https://api.qa.oneid.siemens.com.cn/api/bff/v1.2/developer/ciam/user/profile
在账户登录后，使用 access_token 调用用户信息端点，获取账户基本信息。

SELECT r.roleid, p.permissionid, TRUE
FROM roles r
CROSS JOIN permissions p
WHERE r.rolename = 'Admin'
  AND p.permission IN (
    'device:view',
    'customer:view',
    'project:list',
    'project:create',
    'scheme:view',
    'task:online',
    'task:offline',
    'user:manage',
    'role:manage'
  )
  AND NOT EXISTS (
    SELECT 1
    FROM role_permission rp
    WHERE rp.roleid = r.roleid
      AND rp.permissionid = p.permissionid
  );

    FROM role_permission rp
